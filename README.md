# Drift

> A set of .NET libraries / NuGet packages for message processing and work scheduling.

Drift is a collection of focused, composable libraries. The first package,
[`WindyCliffs.Drift.Abstractions`](src/Drift.Abstractions/README.md), defines the
core abstractions that the rest of the suite builds on.

## Packages

| Package | Description |
| --- | --- |
| `WindyCliffs.Drift.Abstractions` | Core abstractions for message processing and work scheduling. |

## Build & test

```
cd src
dotnet build repo.slnx
dotnet test repo.slnx
```
